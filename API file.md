# 1.用户  

## 1.1. 注册

- POST /User/login  
- payload :  
```json
{  
    "account": "账号",
    "password": "密码"  
}
```
- return :  

```json
{  
    "code": 0,  
    "message": "",  
    "data": null  
}
```

---
## 1.2. 登录
- POST /User/login  
- payload :  
```json
{  
    "account": "账号",
    "password": "密码"  
}
```
- return :  

```json

    "data": null

```

---

## 1.3. 修改自身信息 
- POST/User/changeinfo  
- payload:  
```json
{	  
	"job_id" : "职位号",
	"account":"账号",
	"name":"姓名",
	"telphone":"手机号", 
	 "email":"邮件"  
}
```

- return： 
```json
{  
	"code" : 0,  
	"message": "",  
	"data": null  
}
```

---

## 1.4. 修改密码
- POST/User/changepass 
- payload:  
```json
{	  
	 "oldpass": "旧密码",
	"account" :"账号",
	"newpass" :"新密码"
} 
```
- return：  
```json
{  
	"code" : 0,  
	"message": "",  
	"data": null  
}
```

## 2.3. 项目展示
- POST/User/proshow
- payload:  
```json
{	  
	"pno" : "项目号",
} 
```

```json
{  
	"code" : 0,  
	"message": "",  
	"data":

	"name": "项目名",
	"photo":"项目图片路径",
	"content" : "项目内容",

}
```
